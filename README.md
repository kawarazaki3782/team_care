# TeamCare

「チームケア」は介護者同士や介護者と事業者の交流を目的としたSNSです。<br>
つぶやきの簡単投稿機能や日記の投稿機能を使って外部に発信することができます。<br>
またいいね機能やコメント機能で他者と交流ができます。<br>
<br>
このアプリの目的は介護者を孤立させないことです。<br>
ひとりで悩まず、まずは誰かに相談しましょう。<br>
